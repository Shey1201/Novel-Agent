## 快速入门

### 1. 创建小说

1. 打开应用，点击左侧边栏的"新建小说"按钮
2. 输入小说标题
3. 点击确认创建

### 2. 添加章节

1. 选择要编辑的小说
2. 点击"新增章节"按钮
3. 输入章节标题
4. 开始写作

### 3. 使用Agent生成内容

1. 在编辑器中输入本章大纲
2. 点击顶部工具栏的"运行Agent"按钮
3. 等待Agent协作生成内容
4. 查看生成的文本和溯源信息

## 界面介绍

### 主界面布局

```
┌─────────────────────────────────────────────────────────┐
│  TopBar: 标题 | 写作模式 | 运行Agent | 导出             │
├──────────┬──────────────────────────────┬───────────────┤
│          │                              │               │
│  Sidebar │       Editor                 │  AgentPanel   │
│          │      (Tiptap)                │               │
│  - 小说  │                              │  - Agent消息   │
│  - 章节  │                              │  - 溯源信息    │
│          │                              │               │
│          │                              │               │
└──────────┴──────────────────────────────┴───────────────┘
```

### 左侧面板 (Sidebar)

- **小说列表**: 显示所有小说，支持切换
- **章节列表**: 显示当前小说的所有章节
- **操作按钮**: 新建小说、新增章节、删除等

### 中间编辑区 (Editor)

- **富文本编辑器**: 基于Tiptap的专业写作环境
- **文本溯源**: 选中文本可查看来源Agent和修改历史
- **格式化工具**: 支持基本的文本格式化

### 右侧面板 (AgentPanel)

- **Agent消息流**: 显示Agent执行过程和输出
- **命令输入**: 支持Agent Room命令交互
- **溯源详情**: 显示文本片段的生成历史

### 顶部工具栏 (TopBar)

- **小说标题**: 可点击编辑
- **写作模式**: 切换手动/AI辅助/AI写作模式
- **运行Agent**: 触发多Agent协作流程
- **导出**: 导出为Word文档

## 基础操作

### 小说管理

#### 创建小说

1. 点击左侧"+"按钮或"新建小说"
2. 在弹窗中输入小说名称
3. 可选：添加简介和标签
4. 点击"创建"

#### 编辑小说信息

1. 在小说列表中右键点击小说
2. 选择"编辑信息"
3. 修改标题、简介等内容
4. 点击"保存"

#### 删除小说

1. 在小说列表中右键点击小说
2. 选择"删除"
3. 确认删除操作

**注意**: 删除操作不可恢复，请谨慎操作。

### 章节管理

#### 创建章节

1. 确保已选择小说
2. 点击"新增章节"按钮
3. 输入章节标题
4. 可选：输入章节大纲
5. 点击"创建"

#### 重命名章节

1. 在章节列表中右键点击章节
2. 选择"重命名"
3. 输入新标题
4. 按Enter确认

#### 调整章节顺序

1. 在章节列表中拖动章节
2. 放到目标位置
3. 自动保存新顺序

#### 删除章节

1. 在章节列表中右键点击章节
2. 选择"删除"
3. 确认删除操作

### 内容编辑

#### 基础编辑

- **输入**: 直接在编辑器中输入文字
- **选择**: 鼠标拖动选择文本
- **复制/粘贴**: 支持标准快捷键 Ctrl+C / Ctrl+V
- **撤销/重做**: Ctrl+Z / Ctrl+Y

#### 格式化

| 功能 | 快捷键 | 说明 |
|------|--------|------|
| 粗体 | Ctrl+B | 加粗文本 |
| 斜体 | Ctrl+I | 斜体文本 |
| 下划线 | Ctrl+U | 下划线 |
| 标题 | Ctrl+Alt+1/2/3 | 设置标题级别 |
| 列表 | Ctrl+Shift+8 | 无序列表 |
| 引用 | Ctrl+Shift+B | 引用块 |

#### 文本溯源

1. 在编辑器中选择一段文本
2. 弹出溯源气泡
3. 查看：
   - 来源Agent
   - 生成时间
   - 修改历史

## Agent协作写作

### 写作模式

#### 1. 手动模式

- 完全由作者手动写作
- Agent不参与生成
- 适合：有完整构思的作者

#### 2. AI辅助模式

- 作者主导写作
- Agent提供建议和补充
- 适合：需要灵感和参考的作者

#### 3. AI写作模式

- Agent根据大纲生成完整内容
- 作者进行审核和修改
- 适合：快速生成初稿

### 使用Agent生成章节

#### 准备工作

1. **编写大纲**: 在编辑器中输入本章大纲
   ```
   本章概要：主角在雨夜发现神秘线索
   关键场景：
   - 雨夜街头
   - 废弃钟楼
   - 神秘信件
   ```

2. **设置约束**: 在右侧面板添加写作约束
   - 第一人称视角
   - 悬疑风格
   - 节奏紧凑

3. **配置Agent**: 调整Agent参数（可选）
   - 写作风格
   - 冲突强度
   - 文笔风格

#### 运行Agent

1. 点击顶部"运行Agent"按钮
2. 等待Agent执行（约30-60秒）
3. 查看生成的内容

#### Agent工作流程

```
Planner (规划) 
    ↓
Conflict (冲突分析)
    ↓
Writing (生成初稿)
    ↓
Editor (润色编辑)
    ↓
Reader (读者反馈)
    ↓
Summary (生成总结)
```

### Agent Room命令

在右侧面板的命令输入框中，可以使用以下命令：

#### 基础命令

| 命令 | 说明 | 示例 |
|------|------|------|
| `/start chapter N` | 开始第N章 | `/start chapter 3` |
| `/plan` | 生成写作计划 | `/plan` |
| `/write` | 生成内容 | `/write` |
| `/continue` | 续写内容 | `/continue` |
| `/rewrite` | 重写选中内容 | `/rewrite` |
| `/review` | 审阅内容 | `/review` |

#### 世界观命令

| 命令 | 说明 | 示例 |
|------|------|------|
| `/world` | 查看世界观 | `/world` |
| `/world approve` | 确认世界观 | `/world approve` |
| `/generate outline` | 生成大纲 | `/generate outline` |

#### 示例对话

```
用户: /start chapter 5
Agent: 已加载第5章上下文。根据前4章摘要，建议本章聚焦主角发现真相的关键转折...

用户: /generate outline
Agent: 基于故事发展，建议本章结构：
1. 开场：主角回顾线索
2. 发展：发现关键证据
3. 高潮：与反派对峙
4. 结尾：真相大白

用户: /write
Agent: 正在根据大纲生成内容...
[生成内容显示在编辑器中]
```

1. 点击"时间线"标签

4. 调整事件顺序

### 技能系统

#### 技能分类

- **系统技能**: 内置基础技能
- **写作技能**: 针对写作场景
- **领域技能**: 特定题材专用
- **审核技能**: 质量检查相关

#### 使用技能

1. 打开技能面板
2. 选择要启用的技能
3. 配置技能参数
4. 应用到当前章节

#### 创建自定义技能

1. 点击"新建技能"
2. 填写技能信息：
   - 名称
   - 描述
   - 约束条件
3. 设置目标Agent
4. 保存技能

### 约束配置

#### 全局约束

影响所有Agent的约束：

- **视角**: 第一人称/第三人称
- **时态**: 过去时/现在时
- **风格**: 正式/口语/文学
- **节奏**: 慢速/中等/快速

#### Agent特定约束

针对特定Agent的约束：

- **Planner**: 计划详细程度
- **Writing**: 创意程度
- **Editor**: 编辑重点
- **Reader**: 读者类型

### 导出功能

#### 导出Word

1. 确保当前章节已保存
2. 点击顶部"导出"按钮
3. 选择"导出Word"
4. 下载生成的.docx文件

#### 批量导出

1. 在小说列表中选择小说
2. 右键选择"批量导出"
3. 选择要导出的章节
4. 选择导出格式

## 常见问题

### 编辑器问题

**Q: 内容没有自动保存？**
A: 检查浏览器是否禁用了localStorage，或尝试手动刷新页面。

**Q: 溯源信息不显示？**
A: 确保：
1. 内容是通过Agent生成的
2. 正确选中了文本
3. 后端正常返回trace_data

**Q: 编辑器卡顿？**
A: 尝试：
1. 减少同时打开的章节数量
2. 清除浏览器缓存
3. 关闭浏览器扩展

### Agent问题

**Q: Agent运行失败？**
A: 检查：
1. 后端服务是否运行
2. API密钥是否配置正确
3. 网络连接是否正常
4. 查看浏览器控制台错误信息

**Q: 生成内容质量不高？**
A: 建议：
1. 提供更详细的大纲
2. 添加更多约束条件
3. 调整Agent配置参数
4. 使用Agent Room进行多轮交互

**Q: Agent执行很慢？**
A: 可能原因：
1. LLM API响应慢
2. 网络延迟
3. 生成内容较长

### 数据问题

**Q: 如何备份数据？**
A: 数据保存在 `backend/data/` 目录，定期备份此目录即可。

**Q: 可以导入已有小说吗？**
A: 目前支持通过API导入，未来版本将提供图形化导入功能。

**Q: 数据存储在哪里？**
A: 
- 草稿: `backend/data/<story_id>/drafts/`
- 记忆: `backend/data/<story_id>/memory.json`
- 导出: `backend/data/<story_id>/exports/`

### 性能问题

**Q: 应用加载很慢？**
A: 尝试：
1. 检查网络连接
2. 减少小说/章节数量
3. 使用性能更好的设备

**Q: 导出大文件失败？**
A: 建议：
1. 分批导出章节
2. 检查磁盘空间
3. 增加后端超时时间
